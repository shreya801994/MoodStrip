# MoodStrip – Vintage Photobooth

A browser-based photobooth application that captures multi-frame images and generates a vintage-style photo strip using canvas rendering.

## Features
- Real-time camera capture
- 4-frame photobooth sequence
- Vintage filters (Hollywood, Bollywood, Indie, Golden Hour)
- Canvas-based strip generation
- Downloadable image output
- QR-based sharing

## Tech Stack
- HTML, CSS, JavaScript
- Canvas API
- WebRTC (Camera API)
